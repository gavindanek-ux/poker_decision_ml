# Poker Decision ML

This is a small Python machine learning project that models poker-style decision-making under uncertainty.

## Project idea
I created a simple poker-inspired dataset with variables such as:
- hand strength
- pot size
- bet size
- position

Using this data, I trained a decision tree classifier to predict one of three actions:
- fold
- call
- raise

## Why I made this
I am interested in machine learning, probability, and decision-making under uncertainty. Poker is a natural example of strategic choice with randomness and incomplete information, so I used it as a simple setting for exploring basic ML methods.

## Tools used
- Python
- pandas
- numpy
- scikit-learn

## Files
- `poker_ml.py`: main Python script
- `requirements.txt`: required Python packages

## Notes
This is a beginner project using simulated data and a simple rule-based labeling system. It is meant to demonstrate interest in machine learning and decision modeling, rather than solve poker optimally.
